# ObservePC Ver 2.1

ObservePC は、Windows環境の構成情報を整理して出力する  
インストール不要のシステム情報取得ツールです。

本ツールは診断・修復・最適化を行いません。  
生成AIや技術者が環境を正確に把握するための  
「前提情報」を自動生成することを目的としています。

特に、兼任管理者や小規模環境を担当する管理者が  
生成AIを活用する際の“説明負担”を軽減することを想定しています。

---

## 本ツールの役割

- Windows内部情報の収集
- 構造化されたテキスト出力
- AI相談向け情報整理
- インストール不要（ポータブル実行）

---

## 本ツールが行わないこと

- 自動診断
- 自動修復
- パフォーマンス最適化
- 常駐動作

ObservePCは意図的に  
「事前情報取得」に特化しています。

---

## 想定用途

- ChatGPT/Copilot/Geminiなど生成AIへの環境提示および相談
- Windows構成の記録
- 小規模環境での複数PC管理
- トラブルシューティング前の情報整理

---

## 動作環境

- Windows 11
- Windows 10
- Windows Server 2019

※日本語環境前提

---

## リポジトリ構成

- ObservePC.exe：PC 環境情報取得ツール 実行ファイル
- Source code_2.1.cs.txt：C# による実装コード（Win32 API + WMI） 
- OutputSender.cs.txt：出力ファイルを LAN 経由で送信する参考テンプレートコード  
- ObservePC.cfg.txt:外部設定ファイルのサンプル。使用時はObservePC.cfgへファイル名変更
- README_v2.1.txt：本ファイル   
- DOC_v2.1.txt：詳細説明ドキュメント  
- TECHNICAL_v2.1.txt:外部設定ファイルやソースコード改変など技術情報
- user_note.txt：利用者補足情報（自動取得不可項目・通信スタイル）  
- LICENSE.txt：ライセンス表記  

---

## アーキテクチャ概要

ObservePCは以下の構造で設計されています。

- USBメモリから実行可能な機動性、DLL依存なし
- 複数情報源からの逐次取得・並列取得の選択可能
- 起動オプションと設定ファイルによる柔軟な制御
- AI解釈を意識したテキストでの構造化出力
- コアロジック非改変での拡張可能設計
- 全てのコードを公開している透明性

---

## 設計思想

AI時代において、  
不十分な環境説明は誤った回答を生みます。

ObservePCは  
「AI相談前の問診票」を自動生成するためのツールです。
