# ObservePC Ver 1.9

ObservePC は、Windows 環境の情報を収集・整形し、  
人や AI が状況を正しく理解するための前提情報を提供するツールです。

「その環境では再現しません」  
「情報が足りず、原因が特定できない」

そうしたやり取りを減らすことを目的としています。

---

## 概要

ObservePC は診断ツールではありません。  
問題の原因を自動で特定したり、修正は行いません。

OS・ハードウェア・ネットワーク・セキュリティ設定など、  
Windows 環境に関する情報を広く収集し、  
そのまま AI や人に渡せるテキストとして出力します。

---

## 想定している使い方

ObservePC は、利用者のレベルに応じて使い方を変えられます。

- **初心者**：exe を起動するだけ  
- **中級者**：起動オプションを付けて実行  
- **上級者**：外部ファイルを使って出力を調整  
- **超上級者**：ソースコードを読んで改造

最初は「起動するだけ」で問題ありません。

---

## 典型的な利用シーン

1. ObservePC を実行して環境情報を取得  
2. 出力されたテキストを AI やサポート担当者に渡す  
3. 前提条件が揃った状態で調査・相談を行う  

AI との技術相談や、環境依存トラブルの切り分けを想定しています。

---

## これは何をしないツールか

- 自動診断は行いません  
- 設定変更・修復は行いません  
- 最適化・高速化ツールではありません  

ObservePC は **前提情報を整えること**に特化しています。
